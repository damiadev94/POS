# ADR-004 — Process-Centric Domain

---

**Status:** Accepted

### Context

Al analizar el dominio se observó que las entidades adquieren valor únicamente cuando participan en procesos.

### Decision

El dominio será process-centric.

Las entidades existen para transformarse.

Los procesos representan el comportamiento principal del sistema.

### Consequences

- Information Flow pasa a ser una fase central.
- Los procesos adquieren identidad propia.
- Los estados aparecen como consecuencia de procesos.
