# 06. Information Flow

---

El Information Flow describe cómo las entidades del dominio se desplazan, se transforman y generan valor dentro del Personal Operating System.

Mientras el Domain Model define qué existe y la System Architecture define quién es responsable de cada parte del sistema, el Information Flow explica cómo fluye el trabajo, la información y el conocimiento entre los distintos procesos del dominio.

El foco no está en el movimiento de datos entre herramientas, sino en la evolución de las entidades conceptuales del sistema.
