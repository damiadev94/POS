# POS Filesystem Reference Implementation

Esta es la implementación de referencia del Personal Operating System sobre un sistema de archivos jerárquico.

Su objetivo es validar que el modelo conceptual del POS puede implementarse utilizando únicamente carpetas y archivos.

La arquitectura del POS continúa siendo la fuente de verdad. Esta implementación únicamente describe cómo dicho modelo se materializa en un sistema de archivos.
