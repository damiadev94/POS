# Typography System

## Hierarchy

The typography system is defined in `assets/css/typography.css` and uses a clear visual scale:

- `h1` — page and poster titles
- `h2` — section headers
- `h3` — subsection headings and cards
- `p` — body copy
- `.caption`, `.label` — auxiliary text and status labels

## Examples

- Use `h1` for the main poster title and hero statements.
- Use `h2` to separate major sections such as Architecture, Roadmap and Governance.
- Use `h3` inside cards and sidebars for smaller content units.
- Use `blockquote.quote` for important quotes or executive summaries.

## Good practices

- Keep line lengths around 60–75 characters for readability.
- Use paragraph spacing consistently rather than inline line-height overrides.
- Reserve bold text for emphasis and avoid using multiple weights on a single card.
- Maintain contrast between heading and body text through color and spacing.
