# POS Atlas Poster

This folder contains the starter skeleton for the POS Atlas poster.

## Structure

- `index.html` — the poster shell with placeholder sections for future content.
- `poster.css` — page-specific style overrides for the poster.
- `poster.js` — basic initialization logic and placeholder script.

## Contents

The poster is structured to accept the following sections:
- Purpose
- Vision
- Philosophy
- Principles
- Project State
- Domain Discovery
- Ubiquitous Language
- Domain Model
- Architecture
- Information Flow
- Governance
- Roadmap

## Usage

Copy this structure to create new posters with the same layout and fill each placeholder section with actual content.
