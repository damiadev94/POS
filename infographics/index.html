Reusable poster modules

Every poster is registered through a single configuration object. Use the dashboard to explore available templates and add new projects quickly.

How it works

The system is built with plain HTML, CSS and JavaScript. Templates define a clean poster skeleton, while the design system centralizes colors, spacing and components so every infographic remains consistent.

Ready for expansion

Every poster folder is a self-contained unit. Add new posters by copying the template folder, registering the metadata in assets/js/main.js, and filling the placeholder sections.

Future content

Extend with new infographics

Use the module structure to create architecture dashboards, domain maps, governance diagrams and roadmap posters. The framework is intentionally generic to support high-level technical storytelling.
